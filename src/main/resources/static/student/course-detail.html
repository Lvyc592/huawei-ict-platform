1.1 OSI 与 TCP/IP 协议栈

本章系统讲解 OSI 七层模型与 TCP/IP 协议栈，帮助你建立完整的网络通信认知框架。课程结合华为设备真实配置案例，配合下方实验环境进行实操演练。

本节要点：

  • OSI 七层模型各层职责（物理 → 应用）
  • TCP/IP 四层 / 五层模型对照
  • 数据封装与解封装过程
  • 常见协议：HTTP / TCP / UDP / IP / ARP / DNS

课程完整大纲：

  • 第 1 章 网络基础（45 分钟） · OSI 模型 / IP 地址 / VRP
  • 第 2 章 路由技术（86 分钟） · 静态路由 / OSPF / BGP
  • 第 3 章 交换技术（44 分钟） · VLAN / STP
  • 第 4 章 综合实验（120 分钟） · 企业网组建实操

学员问答（23 条）

  • 学员A：TCP 三次握手中第二次握手时 ACK 和 SYN 能否合并？
    讲师回复：可以合并，称为 SYN+ACK，这是 TCP 协议的标准实现。
  • 学员B：OSI 是理论模型，实际网络都用 TCP/IP，那学 OSI 还有用吗？
    讲师回复：OSI 帮助我们理解"分层思想"，是学习任何协议栈的根基。
